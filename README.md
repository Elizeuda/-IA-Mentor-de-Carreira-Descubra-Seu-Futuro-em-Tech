# 📊 Roadmap de Carreira - Analista de Dados

## 🧩 Visão do Dia a Dia
Como é o trabalho de um(a) **Analista de Dados**:

- Coletar e organizar dados de diferentes fontes  
- Limpar e preparar dados para análise  
- Criar relatórios e dashboards para apoiar decisões  
- Identificar padrões e tendências nos dados  
- Comunicar insights de forma clara para equipes de negócio  

---

## 🧠 Mapa de Skills

**CORE SKILLS (essenciais):**
- Lógica de programação e estatística básica  
- SQL para consultas em bancos de dados  
- Excel/Google Sheets para análises rápidas  

**NICE-TO-HAVE (complementares):**
- Python para análise de dados  
- Visualização avançada com Power BI ou Tableau  

**FERRAMENTAS E TECNOLOGIAS:**
- SQL  
- Python (pandas, numpy, matplotlib)  
- Power BI  

---

## 📅 Roadmap de 90 Dias
**Adaptado para: 14 horas/semana**

### Mês 1 - Fundamentos
**Semana 1-2:**
- Introdução à lógica de programação e estatística básica  
- Conceitos de banco de dados relacionais  

**Semana 3-4:**
- Aprender SQL (SELECT, WHERE, JOIN)  
- Praticar consultas simples em bases abertas  

### Mês 2 - Prática
**Semana 5-6:**
- Introdução ao Python para análise de dados  
- Manipulação de dados com pandas  

**Semana 7-8:**
- Visualização de dados com matplotlib/seaborn  
- Criar pequenos relatórios exploratórios  

### Mês 3 - Portfólio e Preparação
**Semana 9-10:**
- Aprender Power BI (modelagem e dashboards)  
- Criar dashboards simples com dados públicos  

**Semana 11-12:**
- Desenvolver projeto de portfólio completo  
- Simular entrevistas técnicas e comportamentais

## 🚀 Projeto de Portfólio

**Projeto:** Dashboard de Vendas e Tendências  

**O que fazer:**  
Construir um pipeline simples que coleta dados de vendas (dataset público), organiza em SQL/Python e apresenta insights em um dashboard no Power BI.  

**Entregáveis:**
- Base de dados limpa e organizada  
- Relatório exploratório em Python  
- Dashboard interativo em Power BI  

**Critérios de Aceitação:**
- Dados limpos e consistentes  
- Dashboard com pelo menos 3 KPIs (ex.: receita, produtos mais vendidos, sazonalidade)  
- Documentação clara do processo  

**Dica:** Use datasets públicos do Kaggle ou Google Dataset Search para praticar.

## 💬 Roteiro de Entrevistas

**Pergunta 1:** "Você já trabalhou com SQL?"  
**Como responder:** Explique que estudou consultas básicas, cite exemplos de SELECT e JOIN, e mencione projetos práticos.  

**Pergunta 2:** "Como você lida com dados faltantes?"  
**Como responder:** Cite técnicas como remoção de linhas, substituição por média/mediana ou uso de algoritmos que lidam com missing values.  

**Pergunta 3:** "Qual ferramenta de visualização você domina?"  
**Como responder:** Diga que aprendeu Power BI, mostre seu dashboard de portfólio e explique como escolheu os gráficos.  

**Pergunta 4:** "Conte um desafio que enfrentou estudando dados."  
**Como responder:** Relate uma dificuldade inicial (ex.: limpar dados desorganizados) e como resolveu com pandas ou SQL.  

**Pergunta 5:** "Por que você quer crescer como Analista de Dados?"  
**Como responder:** Mostre motivação, fale sobre interesse em inteligência artificial e como vê dados como base para inovação.
